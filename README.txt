Добро пожаловать в Qt-теку!

Смысл “qt-теки” в том, чтобы помочь вам подобрать фильм или сериал под настроение и сделать процесс выбора максимально удобным.
На каждой странице приложения наверху расположены две кнопки (Watch later list и Main window)

Watch later list представляет собой список фильмов, которые вы отметили для себя.
На этой странице можно редактировать этот список или ставить себе напоминание о просмотре на дату.

На основной странице (Main Window) находятся подборки для вас: ... , а также подробная инструкция пользования.

Когда вы запускаете подборку, перед вами появляется интерактивная книга, где страницы - информация о фильме с картинкой.
На любой странице вы можете отметить фильм как нежелаемый или добавить его в список избранных фильмов.



Qt-тека поддерживает взаимодействие с клавиатурой. Список комманд:
	w - Watch later list
	m - Main Window
	i - Instruction
	...
	Режим подборки:
		Left Arrow - вернуться к предыдущему фильму
		Right Arrow - следующий фильм
		Up Arrow - удалить фильм из qt-теки (нежелательный)
		Down Arrow - добавить фильм в Watch later list
